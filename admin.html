Boss Demo Workflow

  1. Open admin.html in a browser.
  2. Click Load topics.json and choose the file from the repo folder.
  3. Add, edit, or delete topics using the form.
  4. Click Save & Download to export an updated topics.json.
  5. Replace data/topics.json in the repo with the downloaded file.
  6. A developer commits and pushes to redeploy via GitHub Actions.
